## 部署
[![Deploy with Vercel](https://vercel.com/button)](https://vercel.com/new/clone?repository-url=https://github.com/jcj-software/comment)

点一下就得，

然后设置环境变量：
|        名称        |                   值                  |
|:------------------:|:-------------------------------------:|
|     REDIS_HOST     |            Redis数据库地址            |
|     REDIS_PORT     |            Redis数据库端口            |
|   REDIS_PASSWORD   |            Redis数据库密码            |

点击部署就行
